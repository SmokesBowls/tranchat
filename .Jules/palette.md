## 2026-03-01 - Missing ARIA Labels on Icon-Only Buttons
**Learning:** Found a persistent accessibility issue across multiple components (`JoinRoom.jsx`, `Room.jsx`, `SetPassword.jsx`) where icon-only buttons (like 'back' and 'exit' navigation buttons) were lacking `aria-label` attributes and visible tooltips, making them completely inaccessible to screen readers and potentially confusing to visually impaired users. Additionally, these buttons lacked visual focus rings for keyboard navigation.
**Action:** When working on navigation or action buttons in this codebase, always check if they are icon-only. If so, ensure they have `aria-label` attributes, `title` tooltips, and appropriate focus visibility classes (e.g., `focus-visible:ring-2 focus-visible:ring-blue-500 focus:outline-none`) using the existing Tailwind configuration.
